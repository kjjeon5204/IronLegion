This package contains 34 different types of steelbeams
and truss-construction elements to build complex structures.
The elements are very low in polygon-count.

All of the elements come in 2 versions. One includes
colliders and a second prefabs without colliders.

The package also contains 19 different texture-sets 
for all of the elements.
Normals maps are also included.
All textures are 1024x1024.
